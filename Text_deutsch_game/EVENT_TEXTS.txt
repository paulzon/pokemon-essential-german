The $player global variable contains information about the player, such as their trainer type, ID number, name, party Pokémon and money.
Die globale Variable $player enthält Informationen über den Spieler, wie z. B. Trainertyp, ID-Nummer, Name, Party-Pokémon und Geld.
This information is in class Player.
Diese Informationen befinden sich in der Klasse Player.
Map connections are how two maps are displayed next to each other in the overworld, allowing you to walk from one to the other.
Durch Kartenverbindungen werden zwei Karten in der Oberwelt nebeneinander angezeigt, sodass Sie von einer zur anderen gehen können.
The connections are defined in the PBS file "map_connections.txt". There is a Debug feature that makes connecting maps easier.
Die Verbindungen werden in der PBS-Datei „map_connections.txt“ definiert. Es gibt eine Debug-Funktion, die das Verbinden von Karten erleichtert.
In the Debug menu, go to "PBS File Editors..." > "Edit map_connections.txt". Instructions are available within. You can move maps around with the mouse.
Gehen Sie im Debug-Menü zu „PBS File Editors...“ > „Edit map_connections.txt“. Anweisungen finden Sie hier. Sie können Karten mit der Maus verschieben.
Every tile in a tileset has a terrain tag, which is a number. This number is used to give certain tiles special effects.
Jede Kachel in einem Kachelset hat ein Gelände-Tag, bei dem es sich um eine Zahl handelt. Diese Zahl wird verwendet, um bestimmten Kacheln besondere Effekte zu verleihen.
For example, ledge tiles have terrain tag 1, which means the player jumps over them. Ledge tiles also have a single passable direction (set separately).
Felsvorsprungplättchen haben beispielsweise die Geländemarkierung 1, was bedeutet, dass der Spieler darüber springt. Ledge-Kacheln verfügen auch über eine einzige begehbare Richtung (separat festgelegt).
Grass tiles (in which you can have wild encounters) have terrain tag 2. Water tiles have terrain tag 6 or 7.
Grasplättchen (auf denen wilde Begegnungen möglich sind) haben die Geländemarkierung 2. Wasserplättchen haben die Geländemarkierung 6 oder 7.
Terrain tag 0 means the tile has no special effect.  See the wiki for more information and a complete list of all terrain tags and their effects.
Geländetag 0 bedeutet, dass die Kachel keinen besonderen Effekt hat. Weitere Informationen und eine vollständige Liste aller Gelände-Tags und ihrer Auswirkungen finden Sie im Wiki.
You can only set terrain tags up to 7 via RPG Maker XP's Database. Essentials has terrain tags with higher numbers.
Sie können über die Datenbank von RPG Maker XP nur Gelände-Tags bis zu 7 festlegen. Essentials verfügt über Gelände-Tags mit höheren Nummern.
To fully set a tileset's terrain tags, open the Debug menu and go to "Other Editors..." > "Edit Terrain Tags".
Um die Terrain-Tags eines Kachelsets vollständig festzulegen, öffnen Sie das Debug-Menü und gehen Sie zu „Andere Editoren …“ > „Terrain-Tags bearbeiten“.

Debug mode is a special way of playing the game. It unlocks various developer features to make it easier to access all parts of the game and test things.
Der Debug-Modus ist eine besondere Art, das Spiel zu spielen. Es schaltet verschiedene Entwicklerfunktionen frei, um den Zugriff auf alle Teile des Spiels und das Testen von Dingen zu erleichtern.
Playing in Debug mode allows you to access the Debug menu (press F9 or find it in the pause menu).
Wenn Sie im Debug-Modus spielen, können Sie auf das Debug-Menü zugreifen (drücken Sie F9 oder finden Sie es im Pause-Menü).
The party screen and Pokémon storage have a similar debug menu for modifying a Pokémon's properties, such as its level or species.
Der Gruppenbildschirm und der Pokémon-Speicher verfügen über ein ähnliches Debug-Menü zum Ändern der Eigenschaften eines Pokémon, wie z. B. seiner Stufe oder Art.
You can use field moves at any time, and hold Ctrl to skip any battles and decide the outcome of Trainer battles. Hold Ctrl while moving to walk over anything.
